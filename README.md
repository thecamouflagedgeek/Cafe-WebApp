This project is a full-stack web application that enables cafés to create, customize, and manage their digital presence.  
The platform allows café owners to maintain their menu, personalize their public page, and track basic order analytics, while customers can view a clean and responsive café menu experience.

The application is built using Next.js with a focus on performance, scalability, and modern UI/UX standards, including support for light and dark themes.

## Features

## Café Owner
- Secure login and signup
- Create and manage café profile (name, logo, description)
- Add, edit, and delete menu items with images and categories
- Owner dashboard for menu, orders, and analytics
- Light and dark theme support

## Customer
- Public café landing page
- Dynamic menu display
- Offers and “What’s New” section
- Mobile-responsive user interface

## MVP Analytics
- Most ordered item
- Least ordered item
